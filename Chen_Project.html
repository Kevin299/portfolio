Project: Mit App Inventor

Course: Computer Science Engineering(CSE)

Duration: 3 weeks

Objective: Create an App for a phone

Type of Project: Pairs

This is the Mit App inventor project that I made with my partner, for the project we were tasked to create an app for a phone. We were given the rubric and instructions to chose a tutorial from Mit App inventor and improve on it. Based on the basic info and code given from the tutorial from Mit App we create a basic quiz app. With the basic code done, we started to change things in the app like new features, and more questions. The point of the quiz app was to test one’s knowledge about the world and random facts. Also we created two different modes, easy and hard, where easy mode is multiple choice and hard mode is you had to type in your answer. Each mode had 20 questions. We added a score to see how well a player would do, and you only have one chance to answer a question. In this project I learend how to use Mit App inventor and the basics of creating apps. I learned how to manage time and how to plan out on finishing a long term project. I was able to commuicate better so we could finish the project on time.
